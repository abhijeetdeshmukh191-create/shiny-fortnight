🚨 AML & Mule Account Investigation Dashboard

📌 Project Overview

This project demonstrates the use of **Excel, SQL, and Power BI** to perform a simulated Anti-Money Laundering (AML) investigation focused on identifying potential mule account activity, high-risk counterparties, suspicious transaction patterns, and geographic risk concentrations.

The project follows a real-world financial crime investigation approach by combining transaction analysis, risk assessment, data visualization, and investigative reporting.

🎯 Business Objective

Financial institutions face increasing risks from:

- Money Laundering
- Mule Account Networks
- Layering of Funds
- High-Risk Counterparties
- Unusual Transaction Patterns

The objective of this project is to identify suspicious activity using data analytics and present actionable insights through interactive dashboards.

# 🚩 AML Rules Implemented

### High Value Transaction Detection

Transactions above predefined thresholds are flagged for review.

### High Risk Counterparty Monitoring

Transactions involving high-risk entities are identified and monitored.

### Geographic Risk Analysis

High-risk cities are mapped to identify possible mule account clusters.

### Transaction Trend Analysis

Transaction behaviour is analysed over time to identify unusual patterns.
